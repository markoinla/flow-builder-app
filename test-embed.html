Flow Builder - Embeddable Viewer Test

This page demonstrates how to embed a flow diagram from Flow Builder into an external website.

How to Test

  1. Start the Flow Builder development server: npm run dev
  2. Open Flow Builder in your browser (usually http://localhost:3000)
  3. Create or open a flow diagram
  4. Click the "Save Flow" button to save it
  5. Click the "Share/Embed" button in the toolbar
  6. Copy the embed code from the dialog
  7. Replace the iframe src below with your flow's embed URL
  8. Open this HTML file in your browser to see the embedded flow
